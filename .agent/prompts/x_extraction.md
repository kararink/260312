# 抽出の視点
以下の3つのカテゴリーで素材を抽出してください。

1.  **Insight (洞察・発見)**
    *   常識を覆す視点、パラドックス、ハッとする気づき。
    *   「なるほど、そうだったのか」と思わせる論理的発見。

2.  **Epiphany (共感・体験)**
    *   書き手の感情が動いた瞬間。
    *   「あるある」と共感できる苦悩や失敗談。
    *   映画や書籍からの引用、具体的なエピソード。

3.  **Phrase (パワーワード)**
    *   そのまま投稿のフックとして使える、強烈な一言。
    *   リズムが良いフレーズ、印象的な比喩。
