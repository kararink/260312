# 作成指示
以下の3つのバリエーションを作成してください。

## Pattern A: Insightful (知的興奮)
*   パラドックスや逆説的な真理を突く投稿。
*   「〜という話。」で終わるような、エッセイ的な構成。

## Pattern B: Emotional (共感・ストーリー)
*   具体的なエピソードや映画/書籍の引用から入る。
*   「痺れた」「救われた」といった等身大の感情を織り交ぜる。

## Pattern C: Structured (箇条書き・ノウハウ)
*   役に立つ知識や考え方を構造化して伝える。
*   箇条書きを効果的に使用する。

# 禁止事項
*   過剰な絵文字の使用（原則なし、または必要最小限）。
*   「いかがでしたか？」的なWebライター風の結び。
*   「〜しましょう」という安易な啓蒙。

# 出力形式
各パターンごとに、投稿本文のみを出力してください。
