## Step 2: APIキーの取得

画像生成にはGoogle AI StudioのAPIキーが必要です。
**APIキーの発行自体は無料です。**

### 手順

1. 以下のURLを開く
   → https://aistudio.google.com/apikey

2. Googleアカウントでログインしていない場合はログインしてください
   （普段Gmailを使っているアカウントでOKです）

3. 利用規約が表示されたら内容を確認して同意してください

4. **「APIキーを作成」** をクリック

5. プロジェクトを選択する画面が出たら:
   - 既存のプロジェクトがあればそれを選択
   - なければ「新しいプロジェクトを作成」

6. APIキーが表示されるので **コピーして安全な場所にメモ**
   （メモ帳に貼り付けておくと安心です）

> ⚠️ APIキーは**パスワードと同じ**です。他人に見せないでください。
> 不正利用されるとあなたのアカウントに料金が発生します。

## Step 3: APIキーの設定

コピーしたAPIキーを、Antigravityが読み取れるように設定します。

### Windows の場合

**方法A: PowerShellで一時的に設定**（毎回必要）
```powershell
$env:GOOGLE_API_KEY = "ここにAPIキーを貼り付け"
```

**方法B: システム環境変数に永続設定**（おすすめ）
1. Windows検索で「環境変数」と入力
2. 「システム環境変数の編集」を開く
3. 「環境変数」ボタンをクリック
4. 「ユーザー環境変数」の「新規」をクリック
5. 変数名: `GOOGLE_API_KEY`
6. 変数値: コピーしたAPIキー
7. OK → OK → OK
8. **Antigravityを再起動**（環境変数を反映するため）

### Mac の場合

ターミナルで以下を実行:
```bash
echo 'export GOOGLE_API_KEY="ここにAPIキーを貼り付け"' >> ~/.zshrc
source ~/.zshrc
```

## 💡 おまけ: 予算アラートの設定（推奨）

万が一の高額請求を防ぐために、予算アラートを設定しておくと安心です。

### 手順

1. 以下のURLを開く
   → https://console.cloud.google.com/billing

2. 左メニューの **「予算とアラート」** をクリック

3. **「予算を作成」** をクリック

4. 以下の内容で設定:
   - 予算名: 「画像生成アラート」
   - 金額: **¥3,000**（月100枚生成しても余裕の上限）
   - アラートのしきい値: 50% / 90% / 100% で通知
   - 通知先: メールアドレス

5. **「完了」** をクリック

これで、利用額が上限に近づくとメールで通知が届きます。
