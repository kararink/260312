## 使い方（3ステップ）

### Step 1: フォルダを配置する
ダウンロードした `ai-image-generator-standard` フォルダを、
あなたのワークスペース（Antigravityで開いているフォルダ）の中にある
`.agent/skills/` フォルダに入れてください。

```
あなたのワークスペース/
└── .agent/
    └── skills/
        └── ai-image-generator-standard/   ← ここに入れる
            ├── SKILL.md
            ├── scripts/
            └── resources/
```

> `.agent` フォルダが見えない場合:
> Windowsではフォルダオプションで「隠しファイルを表示する」をONにしてください。

### Step 2: 以下のプロンプトをコピーしてAntigravityに貼り付ける

---

```
あなたは画像生成のセットアップガイドです。

私はこれからAI画像生成を始めたいです。
以下の手順で進めてください:

1. まず `.agent/skills/ai-image-generator-standard/SKILL.md` を読み込んで、
   このスキルの全体像を理解してください。

2. 次に `resources/setup_guide.md` を読み込んで、
   セットアップの手順を把握してください。

3. `scripts/setup_check.py --help` を実行して使い方を確認してください。

4. 私と一緒に、setup_guide.md の Step 1（料金の説明）から順番に、
   1ステップずつ進めてください。
   各ステップで何をすればいいか、わかりやすく教えてください。

5. 最後に `scripts/setup_check.py --check` を実行して、
   セットアップが完了しているか確認してください。

よろしくお願いします！
```

---

### Step 3: AIの指示に従ってセットアップを進める

プロンプトを貼り付けると、AIがまず **料金の説明** をしてくれます。
納得したら、APIキーの取得と設定に進みます。

全てのチェックが ✅ になれば完了です！
