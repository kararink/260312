## **記事の表現についての法的検討**

### **1\. 厚生年金加入義務の根拠条文**

記事の「厚生年金加入が義務付けられていない小規模企業」という表現は、**法的に正確**です。根拠は以下のとおりです。

---

#### **📌 厚生年金保険法 第6条（強制適用事業所）**

**第6条第1項第1号**（個人事業所の場合）

常時５人以上の従業員を使用する事業所で、**政令で定める業種**に属するもの

→ つまり個人事業所は「**5人以上かつ適用業種**」でなければ強制適用外

**第6条第1項第2号**（法人の場合）

法人事業所は**常時1人以上**使用すれば強制適用

---

#### **📌 非強制適用となるケース（＝「小規模企業」等）**

| 形態 | 条件 | 結果 |
| ----- | ----- | ----- |
| 個人事業主 | 従業員**4人以下** | 強制適用外（任意） |
| 個人事業主 | 5人以上でも**非適用業種**（農林水産業、飲食、理美容、旅館、娯楽等） | 強制適用外 |
| 法人 | 1人以上 | **強制適用**（例外なし） |

非適用業種は**厚生年金保険法施行令 第2条**（健康保険法第3条の読替準用）で列挙されています。

任意加入の規定は同法**第31条**です。

---

#### **✅ 結論**

記事が言う「小規模企業」とは主に：

* **個人事業主で従業員4人以下の事業所**  
* **5人以上でも非適用業種の個人事業所**

これらは厚生年金への加入義務がなく、働く人は「第1号被保険者」として**国民年金＋国民健康保険**に加入することになります。記事の記述は正確です。

---

### **2\. 建設業・宅建業と社会保険加入問題**

#### **🏗️ 建設業の特殊事情**

建設業（土木建築業）は厚生年金の**強制適用業種**に含まれます（令第2条参照）。しかし実態として長年「未加入問題」が深刻でした。

**問題の構造：**

* 建設業は**重層下請構造**が一般的  
* 末端の事業者に一人親方（個人事業主）が多い  
* 一人親方は**被用者ではない**ため、そもそも厚生年金の適用対象外→国民年金のみ  
* 法人であっても小規模・零細は未加入が横行していた

---

#### **📌 国交省による加入義務化の強化（建設業法関連）**

**2020年10月**の建設業法施行規則改正により：

建設業許可の新規申請・更新において、**健康保険・厚生年金・雇用保険への加入**が確認要件となった

未加入の場合、**許可が下りない・更新できない**仕組みに。これは建設業法第7条（許可基準）の「適切な経営」要件として運用されています。

また、国交省は**元請企業に対し、未加入業者・作業員の現場入場を断るよう**指導しており、業界全体での加入率が大幅に改善しています。

---

#### **🏥 建設国保（建設業組合系の健康保険）**

一人親方や小規模事業者が多い建設業界には独自の仕組みがあります：

| 制度 | 内容 |
| ----- | ----- |
| **建設国保**（建設連合国民健康保険組合等） | 国民健康保険組合の一種。組合員とその家族が対象。国保より給付が手厚い場合も |
| **一人親方労災保険** | 特別加入制度（労災保険法第33条以下）。組合経由で加入可能 |
| **年金** | 一人親方は**国民年金のみ**（厚生年金不可）。国民年金基金や小規模企業共済で上乗せ対応 |

建設業組合（全建総連系・全国建設労働組合総連合など）が建設国保の運営母体となっています。

**注意点：** 建設国保に加入していても、**厚生年金の代替にはならない**ため、老後の年金格差問題が依然として存在します。

---

#### **🏠 宅建業者の場合**

宅地建物取引業者については：

| 形態 | 社会保険 |
| ----- | ----- |
| **法人の宅建業者** | 1人から厚生年金・健康保険**強制適用** |
| **個人事業の宅建業者（従業員4人以下）** | 強制適用外→国民年金・国民健康保険 |
| **個人事業（5人以上）** | 不動産業は**強制適用業種**に該当するため強制加入 |

建設業と異なり、宅建業法（宅地建物取引業法）上の**免許要件に社会保険加入は含まれていません**。ただし、国交省の指導・通達により加入促進が図られています。

---

### **まとめ**

記事の表現「義務付けられていない小規模企業」  
　→ 厚生年金保険法第6条に基づき法的に正確

建設業  
　→ 適用業種だが一人親方問題が深刻  
　→ 2020年から建設業許可要件として加入義務が実質強化  
　→ 建設国保で健康保険は対応できるが年金格差は残存

宅建業  
　→ 法人は強制適用、小規模個人事業主は適用外  
　→ 免許要件には社会保険加入なし（建設業と異なる点）
