| 期末自己評価      | 内容                                                                                                                                                                                                                                                                                                                                                                                                                       |
| :---------- | :----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **自己評価**    | **B: 順調に推移 / 継続中**（プロセス重視）                                                                                                                                                                                                                                                                                                                                                                                               |
| **実績・振り返り** | **1. 判定ロジック構造化に向けた「課題の可視化」**<br>建設業法・宅建業法の複雑な許認可要件について、AI実装を見据えた構造化・分解を試行。完全な定義書の完成には至っていないものの（道半ば）、現行フローにおける論理的矛盾や属人化している判断基準を洗い出し、次期システム化に向けた具体的な技術課題を抽出した。<br><br>**2. 小規模プロトタイプによる検証**<br>自らAIツール（LLM/OCR）を用いた小規模なプロトタイプ検証を行い、実務への適用可能性と限界（フィジビリティ）を確認。ベンダー丸投げではない、職員主導のDX推進に必要な基礎知見を蓄積した。<br><br>**3. 総括**<br>「実現可能な落としどころ」を探るプロセスにおいて、技術検証と業務要件の摺り合わせに時間を要している。行政DXの最前線で「何が障壁か」を解明し続けている点は、当初目標に沿った挑戦的行動であったと評価する。 |

## 2. 【管理監督職はマネジメント目標】（チャレンジ面）
**目標**: 業務の効率化と質の向上（重要度の低い業務の廃止・集約、ナレッジ整備）
**期中評価**: システム運用の抜本見直し、内線電話抑止、暗黙知の可視化

| 期末自己評価      | 内容                                                                                                                                                                                                                                                                                                                                                    |
| :---------- | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **自己評価**    | **A: 達成**（組織変革への挑戦）                                                                                                                                                                                                                                                                                                                                   |
| **実績・振り返り** | **1. 業務負荷の「可視化」と「平準化」の断行**<br>特定職員への業務集中という長年の課題に対し、タスクの進捗状況を定量的・定性的に把握する仕組みを導入。柔軟な担当変更や業務分散を果断に実施し、繁忙期においても組織的な遅延・停滞を防いだ。<br><br>**2. ナレッジ共有による「個の負担」軽減**<br>Teamsチャネル等を活用し、法解釈や判断に迷う事例を即座に共有・解決できるフローを定着させた。個人の抱え込みを解消し、心理的安全性の高いチーム運営を実現した。<br><br>**3. 総括**<br>単なる維持管理に留まらず、従来の業務慣習（電話対応や紙ベースの確認等）の抜本的な見直しに踏み込み、持続可能な組織体制への変革（チャレンジ）を推進したことを評価する。 |
